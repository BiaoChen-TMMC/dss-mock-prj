##Brief commands:##

1. configure
`git config`

2. start
 - initialize a project
`
  git init <prj>
`
 - branch from an existing project
`
  git branch <prj>
`
3. pull from github
`
  git pull
`
4. add changes to a commit
`
  git add
`
5. download from remote
`
  git fetch
`
6. commit changes
`
  git commit -m "<comment>"
`
7. other commands
  - git status
  - git show
  - git log
  - git diff
  - git reset
  - git merge
  - git clone
  - git remote add origin <prj>
